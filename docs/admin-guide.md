# Admin Quick Guide

## Login
1. Open /login
2. Enter admin credentials
3. Click Sign in

## Add Member
1. Go to Members
2. Click Add Member
3. Fill name, optional email, and membership end date
4. Upload photo (optional)
5. Save

## Edit Member
1. Go to Members
2. Find the member row
3. Click Edit
4. Update details and save

## Scan Attendance
1. Open Scan QR page
2. Click Start Scan and allow camera permission
3. Point camera at member QR
4. Review displayed details and status
5. Accepted scans are recorded automatically
6. Expired memberships are denied and logged

## Dashboard Monitoring
- View total/active/expired members
- View accepted and denied scan counts for the day
- Review latest scan activity rows

## Settings
- Enable or disable optional photo capture
- Set days-before-expiry reminder threshold

## Operations
- Check storage/logs/app.log for errors
- Run reminder script daily
- Rotate admin password regularly
